Introduction

The most advanced documentation template for your next project.

Welcome to the CustomCSS documentation. This template is designed to be fully responsive, feature-rich, and easy to customize. It includes built-in dark mode and code-copying functionality.

Installation

To get started, install our package via your favorite package manager:

Terminal 
npm install custom-css-toolkit --save-dev

Buttons

Use our custom button styles for actions in forms, dialogs, and more.

HTML 
<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-outline">Outline Button</button>
